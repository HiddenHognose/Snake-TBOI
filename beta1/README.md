# Snaketboi - A Snake's Binding 🐍

A snake-themed roguelike inspired by The Binding of Isaac! Play as a snake fighting off hordes of lizards in dungeon rooms.

## How to Play

### Controls
- **WASD** or **Arrow Keys**: Move your snake
- **Left Click**: Shoot bullets at your mouse cursor
- **ESC**: Quit the game

### Gameplay
- You're a green snake fighting brown lizards
- Click to shoot green bullets at enemies
- Avoid touching lizards - they'll hurt you!
- Defeat all enemies in a room to spawn more
- Survive as long as you can!

### Game Features
- Snake player with shooting mechanics
- Lizard enemies that chase you
- Room-based combat system
- Health system with visual health bar
- Bullet collision detection

## Running the Game

```bash
python3 main.py
```

## Game Structure

- `main.py` - Main game loop and game class
- `player.py` - Snake player class
- `enemy.py` - Lizard enemy class
- `bullet.py` - Bullet/projectile system
- `room.py` - Room/dungeon rendering

Enjoy slithering through the dungeon! 🐍
